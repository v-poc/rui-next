# Skeleton

`Skeleton` screen, generally used to display the loading state of the general structure of the page before the data has been loaded

```jsx pure
<Skeleton.Title />
<Skeleton.Paragraph />
```

## API

### Skeleton / Skeleton.Title

| Properties | Description                                                 | Type                                        | Default |
| ---------- | ----------------------------------------------------------- | ------------------------------------------- | ------- |
| animated   | Whether display the skeleton placeholders loading           | `Boolean`                                   | `false` |

### Skeleton.Paragraph

| Properties | Description                                                 | Type                                        | Default |
| ---------- | ----------------------------------------------------------- | ------------------------------------------- | ------- |
| animated   | Whether display the skeleton placeholders loading           | `Boolean`                                   | `false` |
| lineCount  | The number of paragraph lines                               | `number`                                    | `3`     |
