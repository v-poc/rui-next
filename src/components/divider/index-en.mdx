## API

| Properties   | Description                            | Type                      | Default  |
| ------------ | -------------------------------------- | ------------------------- | -------- |
| contentAlign | The alignment of content               | `center`\|`left`\|`right` | `center` |
| style        | The custom style                       | `CSSProperties`           | -        |
| vertical     | Whether the direction mode is vertical | `boolean`                 | `false`  |
