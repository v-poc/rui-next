## Project setup

Install dependencies

```bash
$ npm i
```

Start the dev server for [`docs-dev`](http://127.0.0.1:5173/rui-next/)

```bash
$ npm start
```

Build UI library

```bash
$ npm run build-lib
```

Build documentation (docs+demos)

```bash
$ npm run build-docs
```
